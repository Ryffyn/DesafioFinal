# Desafio Final

## Introdução
Aplicação criada para o projeto final do EldoTech Training

## Configuração
```
Rode o seguinte comando:
npm install

```

## Base de dados
```
Configure o seu banco no arquivo database.js
npm create-database
npm migrate
npm seed

```


## Rodar projeto
```
npm start

```
## Testes
```
npm test

```
